IIDS Datagrids

Datagrids

Heads up! This example requires AJAX and can't be run locally. Visit GESDH to see it live.

IIDS Datagrids take advantage of the Datatables jQuery plugin.

Datagrids come pre-styled and should fully support the Datatables.net API. This includes loading data with JavaScript Arrays, AJAX and progressive server updates. See datatables.net for more information about datatables configuration.

Description

A Datagrid loading data from the DOM.

Example
Foo Bar Baz Qux Bop
Hello1 Hello2 Hello3 Hello4 Hello5
Snow1 Snow2 Snow3 Snow4 Snow5
Zerp1 Zerp2 Zerp3 Zerp4 Zerp5
Usage

Markup:

<form class="form-search pull-right">
  <div class="input-append">
    <input type="text" class="input-medium search-query" data-filter-table="dt-dom"><button class="btn btn-icon"><i class="icon-search"></i></button>
  </div>
</form>
<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-dom">
  <thead>
    <tr>
      <th class="essential">Foo</th>
      <th class="essential">Bar</th>
      <th class="essential">Baz</th>
      <th class="optional">Qux</th>
      <th>Bop</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Hello1</td>
      <td>Hello2</td>
      <td>Hello3</td>
    </tr>
    <tr>
      <td>Snow1</td>
      <td>Snow2</td>
      <td class="outlier">Snow3</td>
    </tr>
    <tr>
      <td>Zerp1</td>
      <td>Zerp2</td>
      <td>Zerp3</td>
    </tr>
  </tbody>
</table>

Script:

// DOM elements
$('table[data-table-name="dt-dom"]').iidsBasicDataGrid({ 'isResponsive': true });

Notes:

  • Any <input> elements with an attribute data-filter-table="<table-name>" (where <table-name> is the data-table-name of an IIDS data table) will become filters using the standard datatables.net fnFilter() capability.
  • Links are supported transparently, by adding <a href="...">link</a> to the content of a cell.
  • Add class="outlier" to <td> elements to mark a cell as an outlier.
  • To disable the floater on rollover set the useFloater configuration option to false. ex: iidsBasicDataGrid({ useFloater: false });
  • To enable responsive breakpoints set the isResponsive configuration option to true. ex: iidsBasicDataGrid({ isResponsive: true });
  • When isResponsive is enabled, the number of columns displayed in the table depends on the available screen space. By default, a smartphone will display ~2-3 columns, while an expanded desktop browser displays the full set. This is accomplished by assigning semantic classes to the column headings that indicate which data values take precedence (essential vs optional). If a column does not contain an essential or optional class it will always have the lowest precedence.
  • While isResponsive is enabled you can prevent certain breakpoints from firing by using the excludes option. For example excludes: { phone: true }} will prevent the component from dropping optional columns on phones. The supported excludes are phone, portraitTablet and landscapeTablet.

Description

A Datagrid loading data from a JavaScript Array.

Example
Usage

Markup:

<form class="form-search pull-right">
  <div class="input-append">
    <input type="text" class="input-medium search-query" data-filter-table="dt-array"><button class="btn btn-icon"><i class="icon-search"></i></button>
    </div>
 </form>
<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-array"></table>

Script:

// JavaScript array and columns
$('table[data-table-name="dt-array"]').iidsBasicDataGrid({
  'aoColumns': [
    { 'sTitle': 'Schedule', 'sClass': 'essential' },
    { 'sTitle': 'Account Status', 'sClass': 'essential' },
    { 'sTitle': 'DBA Name', 'sClass': 'essential' },
    { 'sTitle': 'Customer Info', 'sClass': 'optional' }
  ],
  'aaData': [
    ['foo', 'bar', 'baz', 'bueller'],
    ['possum', 'raccoon', 'gravy', 'snowflake'],
    ['peanut', 'epsilon', 'flapjack', 'nominee']
  ],
  'isResponsive': true
});

Script:

// Ajax with DataTables, array of objects
$('table[data-table-name="dt-ajax-objects"]').iidsBasicDataGrid({
  'aoColumns': [
    { 'sTitle': 'Rendering engine', 'mData': 'engine', 'sClass': 'essential' },
    { 'sTitle': 'Browser', 'mData': 'browser', 'sClass': 'essential' },
    { 'sTitle': 'Platform(s)', 'mData': 'platform', 'sClass': 'optional' },
    { 'sTitle': 'Engine version', 'mData': 'version' },
    { 'sTitle': 'CSS grade', 'mData': 'grade', 'sClass': 'essential' }
  ],
  'sAjaxSource': '../assets/data/ao.browsers.json',
  'useFloater': false,
  'isResponsive': true
});
